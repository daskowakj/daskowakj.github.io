As part of an effort ot learn Power BI, I completed guided projects supplied by Coursera. The downfall of free access to the Power BI app, is that there is no way to embed the dashboard to an external website because I do not have access to Power BI service.Therefore, the dashboard will be shown as pdf that can be found here.

Getting Started with Power BI Desktop

My Rating 5/5

My first dashboard in PowerBI using data found here. The instructor of the course edited the dataset to randomly distribute the clients across all 50 states of America. The pdf of the dashboard is available here. In this project, data were imported into Power BI Desktop, data were transformed using Power Query and loaded into the workspace. Two slicers are included to show tailored demographic information relevant to default rates across age, sex, education level, and fictional location.

Prepare, Clean, Transform, and Load Data using Power BI

My Rating 5/5

This guided project demonstrated simple ETL processes using Power Query. There was not a publishable deliverable. This guided project was a great hands-on tool for ETL in Power Query.
